2. Define 5 async check functions, each running a specific Cypher query:

Check 1 - Experiments without USES_MODEL:
```cypher
MATCH (e:Experiment) WHERE NOT EXISTS((e)-[:USES_MODEL]->()) RETURN e.exp_id as exp_id, e.status as status
```

Check 2 - Experiments without USES_RECIPE:
```cypher
MATCH (e:Experiment) WHERE NOT EXISTS((e)-[:USES_RECIPE]->()) RETURN e.exp_id as exp_id, e.status as status
```

Check 3 - Stale RUNNING experiments (>24h):
```cypher
MATCH (e:Experiment {status: 'RUNNING'})
WHERE e.created_at < datetime() - duration('PT24H')
RETURN e.exp_id as exp_id, e.created_at as created_at
```

Check 4 - Duplicate config_hash without RETRY_FROM edge:
```cypher
MATCH (e1:Experiment), (e2:Experiment)
WHERE e1.config_hash = e2.config_hash AND e1.exp_id < e2.exp_id
AND NOT EXISTS((e2)-[:RETRY_FROM]->(e1))
AND NOT EXISTS((e1)-[:RETRY_FROM]->(e2))
RETURN e1.exp_id as exp1, e2.exp_id as exp2, e1.config_hash as config_hash
```

Check 5 - Cycle detection in DERIVED_FROM/RETRY_FROM:
```cypher
MATCH path = (e:Experiment)-[:DERIVED_FROM|RETRY_FROM*]->(e)
RETURN COUNT(path) as cycles
```

3. Page layout:
```python
def run() -> None:
    st.title("Admin Console")

    if st.button("Run Check", type="primary"):
        db_client = get_neo4j_client()
        all_passed = True

        for check_name, check_fn in checks:
            with st.expander(check_name, expanded=True):
                results = run_async(check_fn(db_client))
                if not results or (isinstance(results, list) and len(results) == 0):
                    st.success("Passed")
                else:
                    all_passed = False
                    st.warning(f"Found {len(results)} issue(s)")
                    st.dataframe(results)

        if all_passed:
            st.success("All checks passed")
            st.caption("No consistency issues detected.")
```
