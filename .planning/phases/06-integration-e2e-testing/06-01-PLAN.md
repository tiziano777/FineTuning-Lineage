  ```python
  # BEFORE:
  mock_update.assert_called_once_with(
      exp_id="exp-002", status="FAILED", exit_msg="CUDA out of memory",

  # AFTER:

      exp_id="exp-002", status="FAILED", exit_msg="CUDA out of memory", metrics_uri=None,
  )
  ```

  can intercept it.

  root, then call `train(str(test_project / "config.yml"))` instead.

  ```python
  # BEFORE (line ~235):
  def test_callback_injected_when_capture_checkpoints(self, mock_factory, mock_client_cls):
      ...
      result = train("/tmp/config.yml")

  # AFTER:
  def test_callback_injected_when_capture_checkpoints(self, mock_factory, mock_client_cls, test_project):

      ...

  ```

  Do NOT change the patch targets or mock setup.
