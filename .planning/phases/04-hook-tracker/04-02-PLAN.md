# Plan 04-02: Observability Layer (OpenLIT + Unified Metrics)

## Design Decisions

- **Unified metrics**: Single `output.metrics_uri` (JSONL) contains both HW and training metrics
- **In-process SDK**: Training code calls `collector.log_step()` — no sidecar
- **OpenLIT official Docker**: Full stack (UI + OTEL collector + storage) as single service
- **K in decorator**: `@envelope.tracker(collect_every=100)` controls logging frequency
- **Dual-write**: Every log_step → OpenLIT server (indexed by experiment_id) + local JSONL file

## Architecture

```
@envelope.tracker(collect_every=100)
def train(config_path, device):
    collector = get_collector()
    for step in range(total):
        loss = train_step(...)
        collector.log_step(step, {
            "loss": loss,
            "lr": scheduler.get_lr(),
            "gpu_util": get_gpu_util(),    # NVML in-process
            "gpu_mem_used": get_gpu_mem(),
            "gpu_temp": get_gpu_temp(),
        })
        # → internally: if step % K == 0: dual_write()
```

```
[Training Process]

```

### Task 2.2: Create `graph_lineage/observability/collector.py`
```python
class MetricsCollector:
    def __init__(self, experiment_id: str, metrics_uri: str, collect_every: int = 100):
        self._exp_id = experiment_id
        self._metrics_uri = metrics_uri
        self._k = collect_every
        self._buffer: list[dict] = []

    def log_step(self, step: int, metrics: dict) -> None:
        """Buffer metrics. Flush every K steps."""
        entry = {"step": step, "experiment_id": self._exp_id, "ts": now_iso(), **metrics}
        self._buffer.append(entry)
        if step % self._k == 0:
            self._flush()

    def _flush(self) -> None:
        """Dual-write: OpenLIT + local JSONL."""
        self._write_openlit(self._buffer)
        self._write_jsonl(self._buffer)
        self._buffer.clear()

    def _write_openlit(self, entries: list[dict]) -> None:
        """Send batch to OpenLIT via SDK."""
        ...

    def _write_jsonl(self, entries: list[dict]) -> None:
        """Append to metrics_uri file (atomic append)."""
        ...

    def finalize(self) -> None:
        """Flush remaining buffer. Called by POST-execution."""
        if self._buffer:
            self._flush()
```
